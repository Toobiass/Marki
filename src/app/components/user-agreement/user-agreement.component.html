Usage Agreement & Privacy

Before using Marki, please acknowledge the following information regarding privacy and file operations:

Local Data Processing

Marki is an offline-first application. All data (text, Markdown files, and images) is stored exclusively on your local device. No data is transmitted to external servers or cloud services.

File Access

By using Marki, you allow the application to:

  • Create new Markdown files (.md).
  • Read, modify, and overwrite existing Markdown files.
  • Save images to your local file system when pasted.

By clicking "Accept", you confirm that you understand the local-first nature of this application.
